## Pre-Upload Checklist

### Legal & Rights
- [ ] I have the legal right to upload and share this media
- [ ] This media does not contain sensitive or proprietary information
- [ ] This media does not contain personal information (PII)
- [ ] I have reviewed the LICENSE file and agree to the terms

### Technical Requirements
- [ ] File is in the correct directory (`assets/[category]/`)
- [ ] Filename follows naming convention (lowercase, hyphens, descriptive)
- [ ] File has been optimized (see optimization guidelines)
- [ ] File size is within limits (see CONTRIBUTING.md)
- [ ] No duplicate files exist in the repository

### Documentation
- [ ] Commit message is descriptive and clear
- [ ] Pull request includes context about the media
- [ ] Updated relevant README if necessary
