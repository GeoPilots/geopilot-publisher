# Icons

This directory contains icon assets for the GeoPilots project.

## Guidelines

- Use descriptive filenames (e.g., `home-icon.svg`, `settings-icon-24x24.png`)
- Include size information in filename for raster icons
- Prefer SVG format for scalability
- All icon files are tracked with Git LFS

## Organization

Suggested subdirectories:
- `ui/` - User interface icons
- `social/` - Social media icons
- `app/` - Application icons and favicons
