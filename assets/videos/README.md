# Videos

This directory contains video assets for the GeoPilots project.

## Guidelines

- Use descriptive filenames (e.g., `product-demo.mp4`, `tutorial-intro.webm`)
- Compress videos appropriately for web use
- Supported formats: MP4, WebM, MOV, AVI
- All video files are tracked with Git LFS

## Organization

You may create subdirectories to organize videos by type (e.g., demos, tutorials, marketing).
